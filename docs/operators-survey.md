# Обзор операторов из Риги (расширение сверх Join Up)

> Дата: 2026-07-18. Быстрая разведка балтийских операторов на предмет «кого добавить
> вторым источником». Цель — расширить 8 стран Join Up.

## Сводка

| Оператор | Стек | Доступ к данным | Вердикт |
|---|---|---|---|
| **Join Up** (текущий) | Nuxt + открытый REST JSON | Легко, без авторизации | ✅ рабочий источник |
| **Novatours** | Magento 2 + ScandiPWA + GraphQL | Средне-сложно: нужна сессия (`PHPSESSID`) | Размечен, стройка на фазу — см. `novatours-recon.md`. Даёт дальние страны |
| **Tez Tour** | Классика jQuery/PHP, server-rendered | Средне: результаты HTML (`result-box`, `flight-result`) → **скрапинг, хрупко** (риск из SPEC §9) | Возможен как фолбэк, но парсинг разметки |
| **Anex Tour** | Движок **SAMO** + тяжёлый анти-бот-обфускатор | Сложно: страница закриптована | Пропустить пока |
| **Coral Travel** | **PerimeterX** («Pardon Our Interruption») | Сложно: активно блокирует ботов | Пропустить |

## Вывод

**Ни один из остальных не проще Join Up.** Открытый REST JSON у Join Up — везение,
редкость. Из реальных вариантов расширения:

1. **Novatours** — единственный не заблокированный ботозащитой и уже полностью
   размеченный. Но это постройка GraphQL-модуля с сессией (отдельная фаза).
2. **Tez Tour** — запасной, но HTML-скрапинг (хрупко, ровно тот риск, что спека
   помечала). Стек древний (jQuery 2013), значит вряд ли скоро сменят — плюс к
   стабильности парсера.
3. **Anex / Coral** — за анти-ботом (SAMO-обфускатор / PerimeterX). Не стоят драки
   при живом Join Up.

**Замечание:** SAMO и TourVisor — общие движки бронирования у многих операторов.
Если найдётся SAMO/TourVisor-оператор БЕЗ жёсткого анти-бота, один разбор движка
откроет сразу несколько сайтов. Anex на SAMO, но защищён — искать другого на том
же движке имеет смысл, если всерьёз пойдём в расширение.

**Рекомендация:** пока держим Join Up как рабочую лошадку. Расширение = либо стройка
Novatours-модуля (когда захочется дальних стран), либо хрупкий Tez-скрапинг. Быстрой
«ещё-один-REST-как-Join-Up» кнопки нет.
